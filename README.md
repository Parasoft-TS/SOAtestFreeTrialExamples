# SOAtestFreeTrialExamples
Repository containing example test suites for free trial users

## Instructions to import into a clean SOAtest/Virtualize Workspace:
1. Launch SOAtest/Virtualize and point the workspace to a newly created folder.
2. Delete the default ProvisioningAssets, TestAssets, and VirtualAssets projects, including from disk.
3. File > Import > Projects from Git > ... > Import Existing Eclipse Projects > [Select all projects in table]
	- If you wish to edit files from the root directory of the repo that start with '.' then click the 3 dots in Project Explorer > Filters and Customization... > uncheck .* resources.
